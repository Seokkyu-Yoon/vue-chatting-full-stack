# sky-chatting-proto

## backend
### use socket.io to plugin
### express server

### install
```bash
$ npm install
```

### run with development mode
```bash
$ npm run dev
```

### run with product mode
```bash
$ npm start
```

## frontend
### use socket.io-client to plugin
### vue.js

### run with development mode
```bash
$ npm run serve
```
